# ACS-ACR122U-Tool
Python code to read/write nfc card with ACR122U

# Environtment
 * pyscard
 * python 3

# Command List
    $ python nfc.py read
# How to read data
 * Connect the reader to computer and put a card on the reader.
 * Read all data in card with 'read' command
